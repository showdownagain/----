# MT5 Trading System 部署运行说明

本文档用于把当前项目打包到另一台 Windows 电脑或 Windows Server 上运行。

## 1. 目标电脑需要准备

- Windows 10/11 或 Windows Server
- MetaTrader 5 Terminal
- Python 3.11 或 3.12
- 可登录的 MT5 账户

重要说明：

- 后端必须和 MT5 Terminal 安装在同一台电脑上。
- `MetaTrader5` Python 库通过本机 MT5 Terminal 通信，不是直接连接券商服务器。
- 第一版建议只监听 `127.0.0.1`，不要直接暴露公网。

## 2. 打包

在当前电脑项目根目录运行：

```bat
package_release.bat
```

打包结果会生成在：

```text
release\mt5-trading-system-YYYYMMDD-HHMMSS.zip
```

默认不会打包：

- `backend\.env`
- `backend\data\trading.db`
- `backend\data\trading.db-wal`
- `backend\data\trading.db-shm`
- `backend\logs\*.log`
- Python 虚拟环境和缓存

原因是 `.env` 可能包含账号配置，数据库也可能非常大。Tick 数据库主要用于回测，日常看盘不需要跟安装包一起走。

## 3. 在目标电脑安装

1. 解压 zip 到一个固定目录，例如：

```text
D:\mt5-trading-system
```

2. 安装并打开 MetaTrader 5 Terminal。

3. 在 MT5 Terminal 内登录账户，并确认 Market Watch 里有需要的品种，例如 `XAUUSD`。

4. 双击运行：

```bat
install.bat
```

安装脚本会：

- 创建 `backend\.venv`
- 安装 Python 依赖
- 创建 `backend\data` 和 `backend\logs`
- 如果没有 `.env`，从 `.env.example` 复制一份

## 4. 配置

检查并按需编辑：

```text
backend\.env
```

常用配置：

```env
DEFAULT_SYMBOL=XAUUSD
ALLOWED_SYMBOLS=XAUUSD,EURUSD,GBPUSD,USDJPY
ENABLE_REAL_TRADING=false
API_HOST=127.0.0.1
API_PORT=8000
DATABASE_URL=sqlite:///./data/trading.db
```

当前阶段建议保持：

```env
ENABLE_REAL_TRADING=false
```

## 5. 启动

双击运行：

```bat
start.bat
```

启动成功后打开：

```text
http://127.0.0.1:8000/dashboard
```

接口文档：

```text
http://127.0.0.1:8000/docs
```

启动窗口不要关闭。关闭窗口或按 `Ctrl+C` 会停止后端。

## 6. 如果要迁移 Tick 数据库

如果确实要把回测数据库也带到目标电脑：

1. 停止后端。
2. 复制以下文件到目标电脑同名目录：

```text
backend\data\trading.db
backend\data\trading.db-wal
backend\data\trading.db-shm
```

3. 再启动后端。

如果只用于看盘，不需要复制这些数据库文件。

## 7. 云端 Windows Server 建议

如果部署到云端 Windows Server：

- 先只绑定 `127.0.0.1` 本地访问。
- 远程访问建议使用 VPN、Tailscale、ZeroTier 或 Cloudflare Tunnel。
- 不建议把 `8000` 端口裸露到公网。
- 开放公网前必须补登录鉴权、HTTPS、操作日志、风控和一键停止。

## 8. 常见问题

### 后端启动了，但页面显示 MT5 未连接

先确认：

- MT5 Terminal 已打开。
- 已登录账户。
- Python 后端和 MT5 Terminal 在同一台电脑。
- MT5 没有弹出登录、更新或权限提示。

### 页面能打开，但没有 K 线

先打开：

```text
http://127.0.0.1:8000/api/health
```

确认：

- `mt5_connected` 是 `true`
- `tick_sync_running` 是 `true`

再检查：

```text
http://127.0.0.1:8000/api/rates/XAUUSD?timeframe=M5&count=100
```

如果这里报错，通常是 MT5 里没有该品种，或品种名称和券商实际名称不一致。
